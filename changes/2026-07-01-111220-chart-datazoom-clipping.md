# 2026-07-01 11:12:20 图表 X 轴拖动越界修复

## 做了哪些改动

- 将独立图表的 X 轴 `dataZoom` 从 `filterMode: "none"` 改为 `filterMode: "filter"`。
- 为折线、柱状和散点系列显式开启 `clip: true`。

## 为什么做这个改动

- `filterMode: "none"` 会移动坐标窗口但不裁剪窗口外数据，拖动 X 轴时柱状图或点可能渲染到网格外。
- 当前需求是 Y 轴固定、X 轴可移动，因此可以用全量数据计算 Y 轴范围，同时让 X 轴窗口外的数据被过滤掉。

## 解决了什么问题

- 修复拖动 X 轴时数据柱或点跑到图表外的问题。
- 保持每张独立图表仍然可以单独横向拖动。
- 避免图形遮挡坐标轴、图例或相邻图表。
