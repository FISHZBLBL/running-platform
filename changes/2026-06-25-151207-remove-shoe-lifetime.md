# 2026-06-25 15:12:07 +08:00 - 移除跑鞋预计寿命字段

## 改动内容
- 从跑鞋数据结构中移除 `expectedDistanceKm`。
- 从跑鞋校验逻辑中移除预计寿命里程处理。
- 从本地预览模拟 API 中移除预计寿命里程写入。
- 从鞋库添加表单中移除“预计寿命 km”输入框。
- 从鞋库卡片中移除“累计里程 / 预计寿命”的进度条。
- 鞋库卡片现在只显示：
  - 跑鞋照片
  - 跑鞋名称
  - 累计跑量
- 更新跑鞋校验测试。

## 为什么做这个改动
- 当前还没有确认跑鞋寿命应该如何科学定义。
- 不同跑鞋、跑者体重、跑姿、路面、训练强度都会影响寿命，过早加入固定寿命值容易产生误导。
- 当前阶段更适合只做客观记录：这双鞋已经跑了多少公里。

## 解决的问题
- 避免用户误以为平台已经能判断跑鞋寿命。
- 避免后续模型设计前引入不稳定字段。
- 让鞋库第一版保持简单清晰，只承担跑鞋档案和累计里程记录。

## 说明
- 之前的变更记录中保留了当时加入预计寿命的历史描述，本文件记录当前版本已经移除该功能。
- 后续确认寿命规则后，可以重新增加寿命字段、提醒逻辑或基于鞋款/路面/主观感受的寿命模型。
