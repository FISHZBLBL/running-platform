# 2026-07-01 11:08:49 主页趋势图拆分与 X 轴拖动

## 做了哪些改动

- 将主页原来一个 ECharts 容器内的五张图拆成五个独立图表模块。
- 每张图拥有自己的图例、坐标轴、数据窗口和交互状态。
- 为每张图增加 X 轴方向的 `dataZoom`，可以在图表内部横向拖动查看数据。
- Y 轴保留在各自图表内，不再随着整块大图一起横向移动。
- 跑量图的周跑量/月跑量切换按钮移动到跑量图标题行内。

## 为什么做这个改动

- 之前五张图共用一个大画布，手机端左右滑动时整组图表一起移动，查看单独图表不够灵活。
- 第一张图包含多个 Y 轴，和其他图共用横向滚动时更容易造成手机端阅读压力。
- 独立图表可以让每张图按照自己的数据密度控制 X 轴显示范围。

## 解决了什么问题

- 解决五张图被作为一整块横向移动的问题。
- 让每张图都能单独移动 X 轴数据窗口。
- 改善手机端查看大量跑步/体重数据时的可读性。

## 注意事项

- 这次只修改本地代码，未部署到 Netlify。
- 旧的整体图表函数暂时保留但不再调用，便于确认新图表稳定后再清理。
