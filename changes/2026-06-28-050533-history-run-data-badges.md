# 2026-06-28 05:05:33 历史跑步记录数据状态标记

## 做了哪些改动

- 在历史记录的每条跑步记录日期旁增加两个视觉标记。
- 当记录包含分段数据时，显示分段图标和勾选标记。
- 当记录关联跑鞋时，显示跑鞋图标和勾选标记。
- 移除了历史记录里直接显示“跑鞋：...”的文字行，改为图标提示，减少信息噪音。

## 为什么做这个改动

- 历史记录里已经有里程、配速、用时等核心信息，继续用文字写出分段/跑鞋状态会让列表变拥挤。
- 使用图标和勾选可以在不占用太多空间的情况下，让用户快速判断这条记录是否补充了分段和跑鞋信息。
- 手机视口下历史记录空间有限，状态徽标比整行文字更适合扫描。

## 解决了什么问题

- 解决历史记录中无法一眼看出某条跑步是否有关联分段或跑鞋的问题。
- 避免直接显示“有分段记录”“有跑鞋记录”这类文字造成界面重复和臃肿。
- 让跑步记录管理页面的信息层级更清晰。

## 注意事项

- 图标悬停或读屏仍会提供对应信息，方便识别。
- 这次只修改本地代码和记录文件，没有部署到 Netlify。
