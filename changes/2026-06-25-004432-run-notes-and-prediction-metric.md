# 跑步备注与预测卡片调整

- 日期时间：2026-06-25 00:44:32 +08:00
- 分支：`codex/dev-preview`
- 状态：本地预览，未推送、未部署

## 做了哪些改动

1. 从目标预测卡片中移除了“体重-配速相关”指标。
2. 为每条跑步记录增加了 `notes` 字段，用于记录本次跑步的主观感受和训练背景。
3. 在跑步记录表单中增加“主观感受 / 备注”多行输入区域，最多 2,000 个字符。
4. 备注会随跑步记录保存；编辑历史跑步记录时会自动回填。
5. 在服务端校验中将备注设为可选文本，以兼容已有的旧跑步记录。
6. 为备注的保存和去除首尾空白补充了单元测试。

## 为什么做这些改动

- “体重-配速相关”是统计结果，不适合占用目标预测区域的核心位置，且容易被理解为因果关系。
- 跑步表现会受到疲劳、睡眠、天气、身体不适和训练目的等主观因素影响；结构化指标无法完整表达这些背景。
- 先保存主观备注，可以为后续结合文本与跑步数据的预测、训练建议提供可靠的历史数据基础。

## 解决的问题

- 目标预测区域的信息更聚焦于距离、日期和完赛时间。
- 用户可以记录“感觉轻松”“后半程心率偏高”“睡眠不足”等关键信息，不再丢失训练上下文。
- 历史数据与未来备注分析功能之间建立了可持久化的数据通道。

- 在局域网预览中新增、编辑一条跑步记录，确认备注保存并回填。
